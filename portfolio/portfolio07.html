Safari Vs. IE Poster

W3schools.com is an information site for web development. It includes tutorials and references related to web programming. Based on data on web browser usage from W3schools.com, I designed an infographic poster that explored the differences between user trends for Safari (the default browser for Mac) and Internet Explorer (the default browser for Windows).

This academic project focused specifically on statistics of web usage trends of people that visited W3schools.com. The site is for people with an interest in web technology and may not reflect the national or global average. As a result, the concept of the design is related to web programming.
